# Phase 1: Ask

## Business Task

Bellabeat wants to understand how consumers use non-Bellabeat smart devices in order to identify trends that can inform Bellabeat's own marketing strategy, applied to one selected Bellabeat product.

## Guiding Questions

1. What are some trends in smart device usage?
2. How could these trends apply to Bellabeat customers?
3. How could these trends help influence Bellabeat's marketing strategy?
